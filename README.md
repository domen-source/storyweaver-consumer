# StoryWeaver Consumer Website

Customer-facing website for personalized children's books.

## Quick Start
```bash
npm install
npm run dev
```

Opens on http://localhost:3001

**That's it!** Backend is already deployed - no local backend needed.

## For Designers

✅ Safe to change: All styling, layout, text, animations
❌ Don't touch: `/lib/api.ts` (API logic)

Test complete flow before committing:
Upload photos → Avatars → Preview → Payment → Full book
